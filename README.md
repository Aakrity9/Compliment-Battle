# 🌸 Compliment Battle: The Musical Roast 🎶

**Kill them with kindness!** An interactive AI game where your only weapon is your ability to praise. 

## 🎭 The Concept
In most games, you hit enemies. Here, you **praise** them.
* **True Praise:** Lowers the enemy's Ego bar.
* **Mean Comments:** Backfires and hurts your health.

## 🚀 Hyper-Features
- **Gemini AI Engine:** Real-time sentiment analysis of your compliments.
- **Multilingual Moods:** Choose between **Hindi** or **English** music soundtracks that change according to your performance.
- **Dynamic YouTube Integration:** No fixed tracks. The AI searches for the perfect vibe based on the game's situation.

## 🛠️ How to Play
1. Clone the repo.
2. Run `pip install -r requirements.txt`.
3. Run `python main.py` and enter your Gemini API Key.
4. Choose your music language.
5. Start fighting with kindness!
